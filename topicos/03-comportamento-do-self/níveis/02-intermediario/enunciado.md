# Desafio Intermediário

Imagine que você tem uma classe `Configurador` com o método `definir_ambiente`. Se você chamar um método privado da própria classe chamado `log_interno` usando `self`.`log_interno`, isso vai funcionar no Ruby 3.x? E se o método for de uma superclasse? Faça um código que prove o comportamento de métodos privados com o receptor `self` explícito.
