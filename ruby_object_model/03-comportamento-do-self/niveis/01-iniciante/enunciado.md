# Desafio Iniciante

Crie uma classe chamada `Calculadora`. Dentro dela, crie um método de instância chamado `somar` e um método de classe chamado `sobre`. O método `somar` deve retornar o próprio `self` no final (permitindo encadeamento de métodos, ex: `calculadora.somar(5).somar(10)`). O método de classe `sobre` deve *printar* o `self` para provar que ele conhece a própria classe.
